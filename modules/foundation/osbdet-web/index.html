Open Source Big Data Educational Toolkit home page,
the starting point to master Big Data technologies.

Welcome to the OSBDET v2023r1 homepage which will drive you through all the contents in the environment and how to make it work. Please, bear in mind this is an educational environment and it shouldn't be used for production use cases as it doesn't scale to handle large volume of data (all runs in one single node); use it to learn the different frameworks available in the environment or to proof concepts with small datasets.

For more information visit the OSBDET's Github repository.

Jupyter 1.0

Web application used to operate the environment and do all the labs.

NiFi 1.19.1

An easy to use, powerful, and reliable system to process and distribute data.

Hadoop 3.3.1

Framework that allows for the distributed processing of large data sets across clusters of computers using simple programming models.

Spark 3.2.3

Apache Spark™ is a multi-language engine for executing data engineering, data science, and machine learning on single-node machines or clusters.

Kafka 3.3.1

Apache Kafka is an open-source distributed event streaming platform used by thousands of companies for high-performance data pipelines, streaming analytics, data integration, and mission-critical applications.

Truck Fleet Simulator

Data generator creating events related to trucks going over different routes whose drivers break the rules from time to time. Very handy for learning how to code real-time processing jobs.

MariaDB 10.5.18

MariaDB Server is one of the most popular open source relational databases. It’s made by the original developers of MySQL and guaranteed to stay open source. It is part of most cloud offerings and the default in most Linux distributions.

Superset 2.0.1

Superset is fast, lightweight, intuitive, and loaded with options that make it easy for users of all skill sets to explore and visualize their data, from simple line charts to highly detailed geospatial charts.

MongoDB 4.4 CE

The Community version of the MongoDB distributed database offers a flexible document data model along with support for ad-hoc queries, secondary indexing, and real-time aggregations to provide powerful ways to access and analyze your data.

MinIO Dec'22

MinIO offers high-performance, S3 compatible object storage. Native to Kubernetes, MinIO is the only object storage suite available on every public cloud, every Kubernetes distribution, the private cloud and the edge. MinIO is software-defined and is 100% open source under GNU AGPL v3.

Airflow 2.5.0

Airflow is a platform created by the community to programmatically author, schedule and monitor workflows.

Grafana 9.3.2

Query, visualize, alert on, and understand your data no matter where it’s stored. With Grafana you can create, explore, and share all of your data through beautiful, flexible dashboards.

About OSBDET

OSBDET is a test environment creation tool which facilitates the build of sandboxes containing a bunch of open source technologies altogether. These tests environments are targeting people who want to take their first steps with Big Data technologies easily.

OSBDET's architecture encourages the extension of the toolkit by introducing new frameworks with very little effort.
